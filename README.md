# Visual Odometry Research & Notes

This repository focuses on the study, analysis, and reproduction of cutting-edge Visual Odometry (VO) and SLAM algorithms.

## 📝 Research Highlights

### 1. XVO: Generalized Visual Odometry via Cross-Modal Self-Training (ICCV 2023)
- **Core Concept**: Leveraging cross-modal auxiliary tasks (e.g., audio, flow) for self-supervised VO.
- **Key Contribution**: Overcoming the lack of ground truth data and camera intrinsic dependencies.

### 2. ZeroVO: Visual Odometry with Minimal Assumptions (CVPR 2025)
- **Core Concept**: Improving zero-shot generalization through language & geometric guidance.
- **Key Contribution**: Minimal assumption model that generalizes across diverse environments without calibration.
